Cloud Connect WC-1 (WIRELESS ADAPTER) Roland (新品)

■商品紹介

★楽器にRoland Cloudを直結させる革新的なワイヤレス・アダプター WC-1

◎1年間のRoland Cloud Proメンバーシップが付属
Roland Cloud Connect は、Roland Cloudが提供する何千ものサウンドや最新の拡張音源を、対応するハードウェアで即座に使用できるようにします。

1. モバイル・デバイス上のRoland Cloudと対応ハードウェアとをワイヤレスに接続
2. 拡張音源Model ExpansionやWave Expansion、Sound Packをメンバーシップ期間利用可能
3. ワイヤレス・ネットワークを介して、即座に試聴が可能
4. アプリでお好みのModel ExpansionやWave Expansionを選んでタップするだけでハードウェアへのインストールが可能
5. Proメンバーシップで利用可能な楽器全体では30，000以上のサウンドを利用可能
6. JUPITER-X / -Xmで即戦力の9，000以上のサウンドを、サーチ機能とユーザー評価を活用し即座に利用可能
7. Roland Cloud ConnectアプリはAppStore / Google Playから入手可能
8. 1年間のRoland Cloud ProメンバーシップによりRoland Cloudが提供するソフトウェア・コンテンツやハードウェア・コンテンツを利用可能

◎Roland Cloudの豊富なコンテンツが1年間使い放題
Roland Cloud Connect により、ハードウェアまたはソフトウェア・インストゥルメントの30，000を超えるサウンドを利用できる他、Rolandシンセを象徴するModel Expansionなど、Proメンバーシップのすべての機能を1年間無料で使用できます。また、サウンド・エンジンZEN-Coreを心臓部にもつソフトウェア・シンセサイザーZENOLOGY Proをはじめ、本格的なRolandのバーチャル・インストゥルメントのライブラリを入手できる他、ZenbeatsやZentrackerなどの楽曲制作アプリケーションを使用して、どこでも自由に音楽作成も行えます。1年経過後は、必要に応じてメンバーシップを更新したり、Ultimateへアップグレードしたりすることができます。

◎ワイヤレスでの快適な管理操作
Roland Cloud Connect は、ワイヤレス接続でハードウェアへのデータ・インストールをストレスなく行うことができます。付属のワイヤレス・アダプターWC-1をサポートしているRolandハードウェアに接続しiOS / Androidアプリから、Sound PackやWave Expansion、Model Expansionなどお好みのものを選んでインストールするだけです。互換性のあるすべてのコンテンツが検索可能で、Rolandアカウント、およびPC上のRoland Cloud Managerと自動的に同期されます。

◎サウンドのチェックもアプリで簡単
アプリを使用することで個々のトーンやSoundPack、Model Expansionの試聴を即座に行うことができます。探しているものが見つかったら、数秒でハードウェアへインストールすることができます。どこでどんな種類のサウンドを探していても、Roland Cloud Connect がカバーします。
