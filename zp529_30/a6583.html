ローランド Roland Roland Cloud Connect WC-1 ワイヤレスアダプター

Roland Roland Cloud Connect WC-1 ワイヤレスアダプター

楽器にRoland Cloudを直結させる革新的なアダプター
Roland Cloud Connectは、Roland Cloudが提供する何千ものサウンドや最新の拡張音源を、対応するハードウェアで即座に使用できるようにします。

【Standard Features】
・1年間のRoland Cloud ProメンバーシップによりRoland Cloudが提供するソフトウェア・コンテンツやハードウェア・コンテンツが利用可能
・モバイル・デバイス上のRoland Cloudと対応ハードウェアとをワイヤレスに接続
・拡張音源Model ExpansionやSound Packをメンバーシップ期間利用可能
・ワイヤレス・ネットワークを介して、即座に試聴が可能
・アプリでお好みのModel ExpansionsやSound Packsを選んでタップするだけでハードウェアへのインストールが可能
・Proメンバーシップで利用可能な楽器全体では30,000以上のサウンドを利用可能
・JUPITER-X / -Xmで即戦力の9,000以上のサウンドを、サーチ機能とユーザー評価を活用し即座に利用可能
・Roland Cloud ConnectアプリはAppStore / Google Playから入手可能

Roland Cloudの豊富なコンテンツが1年間使い放題
Roland Cloud Connectにより、ハードウェアまたはソフトウェア・インストゥルメントの30,000を超えるサウンドを利用できる他、Rolandシンセを象徴するModel Expansionsなど、Proメンバーシップのすべての機能を1年間無料で使用できます。また、サウンド・エンジンZEN-Coreを心臓部にもつソフトウェア・シンセサイザーZENOLOGY Proをはじめ、本格的なRolandのバーチャル・インストゥルメントのライブラリを入手できる他、ZenbeatsやZentrackerなどの楽曲制作アプリケーションを使用して、どこでも自由に音楽作成も行えます。1年経過後は、必要に応じてメンバーシップを更新したり、Ultimateへアップグレードしたりすることができます。

ワイヤレスでの快適な管理操作
Roland Cloud Connectは、ワイヤレス接続でハードウェアへのデータ・インストールをストレスなく行うことができます。付属のWC-1ワイヤレス・アダプターをサポートしているRolandハードウェアに接続しiOS / Androidアプリから、Sound PacksやWave Expansions、Model Expansionsなどお好みのものを選んでインストールするだけです。互換性のあるすべてのコンテンツが検索可能で、Rolandアカウント、およびPC上のRoland Cloud Managerと自動的に同期されます。

サウンドのチェックもアプリで簡単
アプリを使用することで個々のトーンやSoundPacks、Model Expansionsの試聴を即座に行うことができます。探しているものが見つかったら、数秒でハードウェアへインストールすることができます。どこでどんな種類のサウンドを探していても、Roland Cloud Connectがカバーします。

・ Wi-Fiによるワイヤレス・ネットワーク接続へのアクセス環境（データ送受信料金はユーザー負担となります）
