サージカルマスクで使用される不織布よりも、更に丈夫で高密度な不織布を採用した、品質の高い衛生的な使い捨てマイクカバーです。
マイクヘッド(集音部分)に被せるだけで、飛沫などによる汚染を防止し、マイクを清潔に保ち、異臭の発生も防ぎます。
風防効果により、ハウリングやポップノイズを軽減します。

コロナ禍によるカラオケユーザーの不安を解消する為に、高品質で衛生的な「使い捨てマイクカバー」を取り揃えました。 カラオケ専門店の他、スナック / クラブ / 冠婚葬祭 / 宴会パーティー / 学校 / セミナー / カンファレンス / 講演会 / 各種イベント / 記者会見 / 選挙演説など、マイクを使用する様々なシーンでご活用いただけます。
【装着方法】 1.マイクカバーの開口部下端に親指を入れます。 2.カバーをマイクに当てるように親指で固定します。 3.当てた面が内側に来るようにカバーをマイクに被せます。 4.マイク集音部の全体を覆うように被せたら完成です。
