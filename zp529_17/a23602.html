【施工目安】
5kgで直径20cm・深さ1cmのエグレを5箇所補修可能（施工面積は下地の状態・材質・深さによって異なります）

【硬化時間】
8時間～16時間（気温や湿度によって異なります）

【使い方】
塗り付ける場所のゴミ・ホコリは除去し、充分に乾燥させてください。
1）主剤に硬化剤を入れます。


2）5分間程度、良く混合させます。※混合されると、材料がしっとりとしてきます。混合が不足すると硬化しないことがあります。


3）混合した材料を施工する箇所に配ります。


4）コテで材料をならし、たたくようにして仕上げます。※混合した材料は、30分以内に使用してください。

■印刷工場の出入口段差補修

キレツの発生に伴ないコンクリートが割れ段差ができています

■完成

これで段差の補修は終了です

施工時間：約1時間


【サンケミカル施工事例 FILE 002】

■駐車場の穴・エグレ補修

床にエグレができてしまい台車の通行が困難な状態になっています
転倒事故などが発生する前に床を補修しましょう

■掃除

補修箇所をハケ等で掃除します
汚れやホコリがあると接着力が低下してしまいます

■サンケミカル塗付け

サンケミカルを補修箇所に配ります

コテで平らに仕上げていきます

■完成

これでエグレの補修は終了です
約6時間後には通常業務ができます

施工時間：約30分


【サンケミカル施工事例 FILE 003】

■マンホール廻り破損部分の修繕

マンホール廻りはとても壊れやすいです
モルタルで直してもすぐに左写真のようになってしまいます

コテでたたくようにして埋めていくときれいに仕上がります

■完成

これでエグレの補修は終了です
約6時間後には硬化します

施工時間：約30分
